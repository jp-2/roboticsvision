3D Reconstruction

Last Update 1 day ago5 min read

In this post, I am going to talk about how I was developing and testing an algorithm in Python.

This exercise consists of a 3D reconstruction world of two images with using 2 cameras.
The following paragraphs represents how is the process of the algorithm.

Extract edges

To obtain the edges there are few methods that can be applied, in this case Canny algorithm is choosen.
Once Canny is applied, to get more points represented a morphological operator is applied, in this case, dilate to get thicker points.


Obtain ray

To obtain the backprojection ray is the line that passes through the optical center of the camera
This ray represent the posible positions that a point can be in the 3D representation

Unibotics offer some very useful methods in order to make the algorithm works, for this backproject ray the following API methods where used

HAL.getCameraPosition(...),
HAL.backproject(...),
HAL.graficToOptical(...)


Homologous point using epipolar line

First of all, to get the homologous point of each detected one, it's neccesary to get the epipolar line that goes through it.
To obtain this epipolar line, two points are neccesary. In the case of this algorithm the line is formed by the two more distant points in the line.
Plotting the epipolar line with a fixed thickness, and multiplying it by a image form with 0s, returns a mask which can be use to find the homologous point.

Once we have this mask, by multiplying it by the HSV right image, and using opencv methods like cv2.matchTemplate and cv2.minMaxLoc ,we can obtain a match point from that image.


Least Square

Once we have the point and its match from the other image. we need to find the ray backproject from both.
Using least square we can with np.linalg.lstsq we can minimize the distance of the point from the cameras, and obtain its 3D coordinates


Representing 3D points

With the 3D point obtained with least squares method, the only remain to find is the color of the pixel. This can be done by returning the color of the 2D point we first found.
Using the following method the 3D point can be shown

GUI.ShowNewPoints(...)
